EasyResume

A simple, fast, and free online resume builder. Create a professional resume in minutes and download it as a PDF.

---

## Features

- Clean & responsive design  
- Dark/Light mode toggle  
- Add multiple education & experience entries  
- Real-time preview  
- Download as PDF  
- Works offline (uses local storage)  

---

## How to Use

1. Click **"Start Building Now"**
2. Fill in your details
3. Click **"Save & Preview"**
4. Download your resume as PDF

---

## Tech Stack

- HTML5
- CSS3 (Bootstrap 5)
- JavaScript (Vanilla)
- html2pdf.js (for PDF export)
- Local Storage (no backend)

---

## Files

- `index.html` — Home page
- `builder.html` — Form to enter resume data  
- `preview.html` — Live resume preview + PDF downloads  
- `style.css` — Styles + dark mode  
- `script.js` — Form logic, save, load, PDF
